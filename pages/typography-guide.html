Complete guide to the centralized font classes for consistent typography across all pages

Body Text

.body-lg
Large body text for important content. This is typically used for project descriptions, introductory paragraphs, and other content that needs to stand out while remaining readable.
Font: Crimson Text, Size: 1.2rem, Line height: 1.6
.body-md
Medium body text for regular content. This is the standard size for most text content including project descriptions, list items, and general information throughout the website.
Font: Crimson Text, Size: 1.1rem, Line height: 1.6
.body-sm
Small body text for secondary content. This size is perfect for supporting information, additional details, and content that needs to be less prominent but still readable.
Font: Crimson Text, Size: 1rem, Line height: 1.5

Implementation Guide

How to Update HTML Files:

  1. Replace inline font styles with appropriate classes:
    <h1 class="heading-lg">Page Title</h1>
  2. Update navigation elements:
    <a href="#" class="ui-md">Navigation Link</a>
  3. Apply to content areas:
    <p class="body-md">Content text</p>
  4. Use appropriate intro classes for hero sections:
    <div class="intro-xl">Welcome Text</div>

Responsive Design:

All typography classes automatically adjust for mobile devices. The responsive breakpoint is set at 900px width, where viewport-based sizes scale appropriately and fixed sizes are optimized for smaller screens.
